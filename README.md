### Contributions

This repo comes with a ready made dev environment, using [Vagrant](https://www.vagrantup.com/) and [VirtualBox](https://www.virtualbox.org/). 

Open a shell and execute the following;

```
vagrant up
vagrant ssh
cd /vagrant
npm install
make test
```
